# Sample Bug Reports (Sanitized)

This folder contains a small, sanitized sample of defects reported during the UberEats Web Ordering Platform QA project.

These bug reports were selected to demonstrate:

- Functional UI defect reporting  
- Business logic and risk-based testing  
- Clear reproduction steps and expected vs actual behavior  
- Professional defect documentation style  

## What’s Included

- **Bug 001 – Add to Cart Freezes on Rapid Clicks**  
  Demonstrates UI stability testing and handling of rapid user interactions.

- **Bug 002 – No Quantity Limit for Large Item Quantities**  
  Demonstrates business rule validation, risk assessment, and user safeguard testing.

## Note on Confidentiality

Only a limited number of sanitized bug reports are shared here.  
No internal systems, sensitive data, or proprietary business logic are exposed.

These samples are intended to showcase testing approach, documentation quality, and defect analysis skills.
